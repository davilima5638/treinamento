# Algumas das vantagens do StoryBook

<br />

<b>• Tempo de implementação reduzido - </b> Desenvolva aplicativos de prova de conceito
e prontos para produção mais rapidamente usando componentes predefinidos.
<br />
<br />

<b>• Extensibilidade rápida e simples -</b> Adicione recursos avançados como indicadores
de digitação, recibos de leitura, reações etc. sem escrever códigos complexos.
<br />
<br />

<b>• Componentes flexíveis e personalizáveis - </b> Personalize o design de componentes
e adicione componentes personalizados para estender a funcionalidade.
<br />
<br />

<b>• Tema fácil -</b> Use os temas claros e escuros integrados ou crie temas personalizados
para vários casos de uso: bate-papos em grupo, suporte e eventos.
